# Bing Daily Wallpaper Downloader 🖼️

Una herramienta sencilla y ligera para obtener el fondo de pantalla diario de Bing en su máxima resolución directamente en tu PC.

## ✨ Características
* **Máxima Calidad:** Consulta la API oficial de Bing para descargar la imagen en la mejor resolución disponible.
* **Fácil de usar:** No requiere instalación de Python ni dependencias externas.
* **Portable:** Ejecutable `.exe` de un solo clic.

## 🚀 Uso rápido
1. Ve a la carpeta raíz del proyecto [bingWallpapers](github.com/juanillufriu/bingWallpapers/).
2. Descarga el archivo `bing_wallpaper.exe`.
3. Ejecútalo y la imagen se guardará automáticamente en una carpeta BingWallpapers (se encuentra en la misma ruta donde tengas tu .exe descargado).

## 🛠️ Tecnologías utilizadas
* Python (Backend del script)
* Requests (Para consultas a la API)
* PyInstaller (Para la creación del ejecutable)
